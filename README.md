# U.S. Labor Market: 2020-Present
This repository is for practicing how to create an interactive Streamlit dashboard for visualizing data series from the US Bureau of Labor Statistics.
This project will utilize the following four data series from the US Bureau of Labor Statistics (BLS) to analyze how key aspects of the U.S. labor market were influenced by the COVID-19 pandemic and how they have recovered in the years that followed:
- Unemployment Rate
- Total Nonfarm Employment
- Total Private Average Weekly Hours of All Employees
- Total Private Average Hourly Earnings of All Employees
